Agent 意图识别完全指南：从规则匹配到 LLM 的 5 种实战方案

2026-03-19 14:35·全栈直男

摘要：意图识别是 AI Agent 的"大脑中枢"，决定了 Agent 能否准确理解用户需求。本文深度解析 5 种主流意图识别方案，附带完整代码示例和性能对比，助你打造更聪明的 AI 助手。

一、为什么意图识别是 Agent 的核心能力？

想象一下这个场景：

用户："帮我查一下北京明天的天气，然后发给老王"

一个优秀的 Agent 需要瞬间完成以下判断：

  • 核心意图：查询天气 + 发送消息（复合意图）
  • 槽位提取：地点=北京、时间=明天、收件人=老王
  • 任务路由：调用天气 API → 调用消息发送工具
  • ⚠️ 边界识别：这不是闲聊，需要执行具体任务

意图识别的质量直接决定了 Agent 的智能程度。识别错误会导致：

  • ❌ 调用错误的工具
  • ❌ 遗漏关键参数
  • ❌ 给出答非所问的回复

三、方案一：规则匹配（最快上手）

核心思路

通过关键词、正则表达式匹配用户输入，适合指令型场景。

代码实现

import re
from typing import Dict, Optional

class RuleIntentRecognizer:
    """基于规则的意图识别器"""
    
    def __init__(self):
        self.intent_patterns = {
            "query_weather": [
                r".*天气.*",
                r".*气温.*",
                r".*下雨.*|.*下雪.*",
            ],
            "set_reminder": [
                r".*提醒.*",
                r".*记得.*",
                r".*别忘.*",
            ],
            "search_web": [
                r".*搜索.*",
                r".*查一下.*",
                r".*百度.*|.*谷歌.*",
            ],
            "play_music": [
                r".*播放.*音乐.*",
                r".*听歌.*",
                r".*来首.*",
            ],
        }
    
    def recognize(self, text: str) -> Dict:
        """识别用户意图"""
        for intent, patterns in self.intent_patterns.items():
            for pattern in patterns:
                if re.match(pattern, text, re.IGNORECASE):
                    return {
                        "intent": intent,
                        "confidence": 0.8,  # 规则匹配固定置信度
                        "matched_pattern": pattern
                    }
        return {"intent": "unknown", "confidence": 0.0}

# 测试
recognizer = RuleIntentRecognizer()
print(recognizer.recognize("北京明天会下雨吗"))
# 输出：{'intent': 'query_weather', 'confidence': 0.8, 'matched_pattern': '.*下雨.*'}

四、方案二：传统 ML 分类（性价比之选）

核心思路

将意图识别建模为文本分类问题，使用 SVM、FastText 等模型。

代码实现（使用 sklearn）

from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.svm import SVC
from sklearn.pipeline import Pipeline
import joblib

class MLIntentRecognizer:
    """基于机器学习的意图识别器"""
    
    def __init__(self):
        # 构建 Pipeline：TF-IDF + SVM
        self.pipeline = Pipeline([
            ('tfidf', TfidfVectorizer(
                ngram_range=(1, 2),  # 使用 unigram 和 bigram
                max_features=5000,
                min_df=2
            )),
            ('svm', SVC(
                kernel='rbf',
                C=10,
                probability=True
            ))
        ])
    
    def train(self, texts, labels):
        """训练模型"""
        self.pipeline.fit(texts, labels)
        print(f"训练完成，样本数：{len(texts)}")
    
    def predict(self, text: str) -> Dict:
        """预测意图"""
        intent = self.pipeline.predict([text])[0]
        probs = self.pipeline.predict_proba([text])[0]
        confidence = max(probs)
        
        return {
            "intent": intent,
            "confidence": float(confidence),
            "all_scores": dict(zip(self.pipeline.classes_, probs))
        }
    
    def save(self, path: str):
        joblib.dump(self.pipeline, path)
    
    def load(self, path: str):
        self.pipeline = joblib.load(path)

# 训练数据示例

五、方案三：语义向量匹配（冷启动神器）

核心思路

使用预训练 Embedding 模型将文本编码为向量，通过相似度匹配找到最接近的意图。

核心优势

  • 零样本/少样本：每个意图只需 1-3 个示例
  • 语义理解：能识别"查天气"和"天气怎么样"的相似性
  • 快速上线：无需训练，直接可用

代码实现（使用 sentence-transformers）

from sentence_transformers import SentenceTransformer
import numpy as np
from sklearn.metrics.pairwise import cosine_similarity

class EmbeddingIntentRecognizer:
    """基于语义向量的意图识别器"""
    
    def __init__(self, model_name="BAAI/bge-small-zh-v1.5"):
        # 使用中文预训练模型
        self.model = SentenceTransformer(model_name)
        self.intent_examples = {}  # {intent: [example_texts]}
        self.intent_embeddings = {}  # {intent: embedding}
    
    def add_intent(self, intent: str, examples: list):
        """添加意图示例"""
        self.intent_examples[intent] = examples
        # 计算示例的平均向量作为该意图的代表向量
        embeddings = self.model.encode(examples)
        self.intent_embeddings[intent] = np.mean(embeddings, axis=0)
        print(f"✅ 添加意图：{intent} ({len(examples)} 个示例)")
    
    def recognize(self, text: str, top_k: int = 3) -> Dict:
        """识别意图"""
        # 编码用户输入
        query_embedding = self.model.encode([text])[0]
        
        # 计算与各意图的相似度
        scores = {}
        for intent, embedding in self.intent_embeddings.items():
            similarity = cosine_similarity(
                [query_embedding], 
                [embedding]
            )[0][0]
            scores[intent] = similarity
        
        # 排序
        sorted_intents = sorted(scores.items(), key=lambda x: x[1], reverse=True)
        
        return {
            "intent": sorted_intents[0][0],
            "confidence": float(sorted_intents[0][1]),
            "top_k": [
                {"intent": i, "score": float(s)} 
                for i, s in sorted_intents[:top_k]
            ]
        }

# 使用示例
recognizer = EmbeddingIntentRecognizer()

# 每个意图只需少量示例
recognizer.add_intent("query_weather", [
    "查询天气",
    "明天会下雨吗",
    "北京气温多少度",
])

六、方案四：LLM Prompt（当前最热门）

核心思路

利用大语言模型的强大理解能力，通过精心设计的 Prompt 实现意图识别。

核心优势

  • 最强泛化能力：理解各种表达方式
  • 零样本识别：无需示例即可工作
  • 可解释输出：可要求模型给出推理过程

代码实现（使用 OpenAI 兼容 API）

from openai import OpenAI
import json

class LLMIntentRecognizer:
    """基于大语言模型的意图识别器"""
    
    def __init__(self, api_key: str, base_url: str = None):
        self.client = OpenAI(
            api_key=api_key,
            base_url=base_url or "https://api.openai.com/v1"
        )
        
        # 定义意图 schema
        self.intent_schema = {
            "query_weather": {
                "description": "查询天气、气温、降水等气象信息",
                "slots": ["location", "date"],
                "tools": ["weather_api"]
            },
            "set_reminder": {
                "description": "设置提醒、闹钟、待办事项",
                "slots": ["time", "content", "method"],
                "tools": ["reminder_service"]
            },
            "search_web": {
                "description": "搜索网络信息、查询资料",
                "slots": ["query", "source"],
                "tools": ["search_engine"]
            },
            "chat": {
                "description": "闲聊、问答、不涉及具体任务",
                "slots": [],
                "tools": []
            },
        }
    
    def recognize(self, text: str) -> Dict:
        """识别用户意图"""
        
        prompt = f"""你是一个意图识别专家。请分析用户输入的意图。

## 可用意图列表
{json.dumps(self.intent_schema, ensure_ascii=False, indent=2)}

## 输出要求
1. 从上述意图中选择最匹配的一个
2. 提取相关的槽位信息
3. 给出置信度 (0-1)
4. 输出 JSON 格式

## 用户输入
{text}

## 输出
"""
        
        response = self.client.chat.completions.create(
            model="gpt-4o-mini",  # 或 DeepSeek-V3、通义千问等
            messages=[
                {"role": "system", "content": "你是一个精准的意图识别助手，只输出 JSON。"},
                {"role": "user", "content": prompt}
            ],
            temperature=0.1,  # 低温度保证输出稳定
            response_format={"type": "json_object"}
        )
        
        result = json.loads(response.choices[0].message.content)
        return result

# 使用示例（以 DeepSeek 为例）
recognizer = LLMIntentRecognizer(
    api_key="your-deepseek-api-key",
    base_url="https://api.deepseek.com/v1"
)

]

for text in test_cases:

    print(f"输入：{text}")
    print(f"意图：{result}")
    print("-" * 50)

九、实战：混合方案架构设计

生产环境中，推荐采用分层识别架构：

    │
    ▼
┌─────────────────┐
│  Layer 1: 规则   │ ← 快速匹配高频指令（<1ms）
└────────┬────────┘
         │ 未匹配
         ▼

代码实现

class HybridIntentRecognizer:
    """混合意图识别器"""
    
    def __init__(self, rule_recognizer, embedding_recognizer, llm_recognizer):
        self.rule = rule_recognizer
        self.embedding = embedding_recognizer
        self.llm = llm_recognizer
        
        self.embedding_threshold = 0.75  # 向量匹配阈值
    
    def recognize(self, text: str) -> Dict:
        # Layer 1: 规则匹配
        rule_result = self.rule.recognize(text)
        if rule_result["intent"] != "unknown":
            rule_result["layer"] = "rule"
            rule_result["latency"] = "<1ms"
            return rule_result
        
        # Layer 2: 向量匹配
        embedding_result = self.embedding.recognize(text)
        if embedding_result["confidence"] > self.embedding_threshold:
            embedding_result["layer"] = "embedding"
            embedding_result["latency"] = "~100ms"
            return embedding_result
        
        # Layer 3: LLM 识别
        llm_result = self.llm.recognize(text)
        llm_result["layer"] = "llm"
        llm_result["latency"] = "~500ms"
        return llm_result

十、避坑指南：5 个常见陷阱

陷阱 1：意图粒度过细

❌ 错误：query_weather_beijing, query_weather_shanghai, ...
✅ 正确：query_weather + slot: location

陷阱 2：忽略"其他"意图

# 必须处理无法识别的情况
if confidence < threshold:
    return {"intent": "unknown", "fallback": "clarify"}

陷阱 3：训练数据偏差

❌ 只有正面样本："查询天气"
✅ 包含变体："天气咋样"、"会下雨不"、"气温多少"

陷阱 4：忽视置信度校准

# 不要盲目相信模型输出
if result["confidence"] < 0.6:
    # 请求用户澄清
    ask_for_clarification()
